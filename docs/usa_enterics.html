dataset: United States enteric pathogen multiplex antibody data

Usage

data(usa_enterics)

Format

A data frame with 86 observations and 10 variables.

id
individual ID

age
age in years. Note that age records years completed, except for children <1 year old.

cp17
response (MFI-bg) to Cryptosporidium parvum recombinant 17-kDa antigen

cp23
response (MFI-bg) to Cryptosporidium parvum recombinant 27-kDa antigen

vsp5
response (MFI-bg) to the VSP-5 fragment of Giardia intestinalis variant-specific surface protein 42e

leca
response (MFI-bg) to Entamoeba histolytica lectin adhesion molecule (LecA)

etec
response (MFI-bg) to Enterotoxigenic Escherichia coli (ETEC) heat labile toxin β subunit

salb
response (MFI-bg) to lipopolysaccharide (LPS) from Salmonella enterica serotype Typhimurium (Group B)

norogii
response (MFI-bg) to purified recombinant norovirus GII.4 New Orleans virus-like particles from a baculovirus expression system

norogi
response (MFI-bg) to purified recombinant norovirus GI.4 virus-like particles from a baculovirus expression system

Description

IgG antibody responses to protozoan, bacterial, and viral enteric pathogens measured with a multiplex bead assay in sample sets collected from 86 anonymous blood donors in the USA. There is one record per child. All antibody levels were measured on the Luminex platform and are in units of median flourescence intensity minus background (MFI-bg). A comparable dataset (and run with the same bead set) for children in Haiti is also included in the package: haiti_enterics.
